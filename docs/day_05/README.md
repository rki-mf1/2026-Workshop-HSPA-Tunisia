# Day 05 — Working on Your Own Data

## Overview

Day 05 is dedicated to applying the concepts and tools learned during the workshop to your own sequencing data.

Participants will work individually or in small groups on datasets prepared before the workshop. Instructors will be available to support data organization, quality control, mapping, reference-based assembly, variant calling, and interpretation of results.

## 🎯 Learning goals

By the end of Day 05, you should be able to:

- organize your own sequencing data in a reproducible project structure
- choose an appropriate analysis strategy for your dataset
- perform read quality control
- run assembly workflows
- inspect key output files and quality metrics
- interpret results and identify potential problems
- document the main analysis steps used for your own data

## 💻 Practical work

During this day, you will apply the workshop workflow to your own dataset.

Suggested workflow:

```text
own Illumina reads
  ↓
data organization
  ↓
De novo assembly
  ↓
cgMLST and/or IQ-TREE
  ↓
Result interpretation
```

## 🧪 Working session

Participants should work on their own prepared Illumina short-read data.

Please make sure you know:

- where your raw FASTQ files are located
- what organism or pathogen your data comes from
- whether a suitable cgMLST scheme is available
- which analysis questions you want to answer
- which outputs you would like to generate by the end of the day
